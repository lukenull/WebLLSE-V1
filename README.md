# WebLLSE (Web-based Low-Level Synthesizer Experiment)

## Description
WebLLSE is a web-based audio generator that which uses Web Audio API to generate synthesized electronic sounds based on the provided data. It differs from most synth virtual instruments in that its generated sounds are based on hard-coded math expressions, which allows a high level of flexibility and low-level control over the sound.

There is no required installation or setup; this program runs directly in the browser.
