Design and Musical Tips
Transposition

You probably already know that using $F# as a frequency parameter will set its frequency to the currently playing note.
To transpose a wave up or down, use $F#*(2**(n/12)), replacing n with the number of semitones you want to transpose, positive for up and negative for down. For example, if you set frequency to $F#*(2**(1/12)), you will hear a C# when you press the C key. To increase frequency by some number of semitones, the new frequency will be $$ F\cdot\left(2^{\frac{n}{12}}\right) $$ where F is the original frequency and n is the number of semitones changed.
You could also use the shorthand, utilizing the $smt# global variable, with frequency as $F#*($smt#**n)
Creating Common Synth Sounds
  • Supersaw: This is the heavy, layered, bright sound very commonly used in electronic music. It is composed of several (7-16) sawtooth waves stacked on top of each other and detuned slightly, usually by around 5-20 semitones.
    To create this, add a sawtooth wave with a modifier. Decide how many sawtooth waves you want, and set the end value to that number, with start at 0 and step at 1. We will reference the variable as a. Set frequency to $F#*($smt#**($a#*0.10)), which will go through each saw wave and detune it by 10 cents per its index.
    This would sound thin by default because everything is centered. Set the panning to -1+(2 * ($a# % 2)) to alternate each wave between far left and right.
